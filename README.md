# AI Team Orchestrator

Multi-AI collaboration platform that orchestrates OpenAI GPT, Google Gemini, and Anthropic Claude through a unified web interface for comparative analysis and task delegation.

## Features

- Unified interface for GPT, Gemini, and Claude
- Side-by-side response comparison
- Task routing to the best-fit model
- Conversation history across all models
- Response quality scoring

## Tech Stack

JavaScript, Express, OpenAI API, Gemini API, Anthropic API

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
git clone https://github.com/karthik-idikuda/team.git
cd team
npm install
```

### Usage

```bash
npm start
```
